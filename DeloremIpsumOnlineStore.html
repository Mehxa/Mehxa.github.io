Project Description:

For this project, our team was tasked to create a web application to solve a problem.

We were given the theme : "eCommerce: How might we use IT to improve inventory management systems"

We developed an online pharmacy store website which provides the typical features of an eCommerce website but uses abstraction for more efficient storage management.

My Contributions:

  • Created multiple pages, product display design and the navbar
  • Developed functions for the addition and deletion of products when shopping
  • Developed the necessary functions for shopping transactions
  • Implemented functions to allow users to edit their transactions or cancel their orders
