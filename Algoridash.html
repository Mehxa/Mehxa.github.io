Project Description:

For this project, I was tasked to create a Python application with learning materials to enhance the learning of selected cryptography topics. I chose to do a desktop application as I had no prior experience in doing so and was told that Tkinter was easy to learn.

Features:

  • A cryptography tool that allows the user to select from multiple encryption concepts and techniques
  • A brief lesson and quiz for a topic that I was required to teach through the application

Cryptographic Techniques Covered

  • Shift (Caesar) Cipher
  • Mono-alphabet Cipher
  • Rail Fence Technique
  • Simple Columnar Transposition Technique
  • Vernam Cipher
  • Diffie-Hellman Key Exchange
  • AES | ECB, CBC, CFB, OFB | 128, 192 & 256 bits

I was also required to create an installation guide and a user guide for the application.
